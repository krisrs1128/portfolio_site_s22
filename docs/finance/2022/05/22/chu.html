IMDB Top 1000 Movies

Author: Sirui Chu

Click here to visit the interactive app.

The portfolio is an interactive interface that allows users to examine the relationship between the numeric variables in the top 1000 IMDB movies dataset from Kaggle. The users can select movies with specific genres. Also, it allows the users to choose the numeric variables plotted on the axes and also shows the distribution of the certificate of the movies selected.
