Nobel Prize Winners

Author: Jonah Maroszek

Read the portfolio by clicking here.

This project aims to find patterns in past Nobel Prize winners. Some variables that were explored include birth country and the age they were awarded their prize. This project includes a spatial visualization and a ridge plot visualization to help find out more about these key variables.
